# 📈 StockVision - Market Predictor

StockVision is a DAA-based stock market prediction web application that analyzes stock price trends using the Sliding Window Algorithm.

The project allows users to:
- Predict stock prices
- Analyze market trends
- View stock dashboard data
- Understand algorithm-based prediction systems

Built using Node.js, Express.js, HTML, CSS, and JavaScript. :contentReference[oaicite:0]{index=0}

# 🚀 Features

- 📊 Stock Price Prediction
- 📈 Trend Analysis (Uptrend / Downtrend)
- ⚡ Sliding Window Algorithm
- 📋 Interactive Dashboard
- 🔍 Search Stocks Feature
